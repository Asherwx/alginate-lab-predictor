# 海藻酸钠/壳聚糖复合材料调控藻液 pH 实验研究方案

## 1. 研究定位

本课题可定位为“环境功能材料与藻类响应的耦合调控”。核心问题不是单纯比较不同 pH 下 OD 的变化，而是建立以下因果链：

材料配方与制备条件 -> 复合材料结构和表面电荷 -> 不同 pH 下与藻液/污染物的相互作用 -> OD、去除率或抑制率变化 -> 机器学习预测与实验优化。

适合的研究对象包括：

- 藻液生长/抑制响应：以 OD680 或 OD750 表征藻细胞密度变化。
- 污染物去除体系：若藻液中加入氮、磷、染料、重金属或抗生素，可预测去除率、吸附量或毒性缓解效果。
- 材料稳定性：评价复合材料在不同 pH 下的溶胀、质量损失、机械完整性和重复使用性。

## 2. 推荐实验因素

建议优先记录这些变量，后续建模才有足够信息：

| 类别 | 变量 | 建议范围 |
|---|---|---|
| 溶液配方 | 海藻酸钠浓度 | 0.5%-3.0% |
| 溶液配方 | 壳聚糖浓度 | 0.2%-2.0% |
| 凝胶/培养环境 | 冰乙酸体积分数或乙酸蒸汽暴露时间 | 0.5%-5%，或 2-24 h |
| 改性 | 改性剂类型 | CaCl2、Fe3O4、PDA、金属盐、生物炭等 |
| 改性 | 改性剂浓度 | 按实际体系设置 |
| 干燥 | 冷冻温度、冻干时间 | -20 至 -80 degC，12-48 h |
| 藻液 | 初始 pH | 4-10 |
| 藻液 | 初始 OD | 0.05-1.0 |
| 反应条件 | 材料投加量 | 0.1-10 g/L |
| 反应条件 | 振荡转速 | 80-200 rpm |
| 反应条件 | 培养/反应时间 | 2-120 h |
| 响应指标 | 终点 OD、去除率、pH 变化 | 按实验目标选择 |

## 3. 建议分阶段设计

### 阶段 A：材料制备和表征

1. 设置海藻酸钠/壳聚糖质量比，例如 3:1、2:1、1:1、1:2。
2. 在冰乙酸环境中培养或交联，控制暴露时间。
3. 改性方向可选：
   - Ca2+ 交联增强结构稳定性。
   - Fe3O4 赋磁，便于分离回收。
   - 多巴胺/PDA 增加表面活性位点。
   - 生物炭或矿物粉体提高孔结构和吸附位点。
4. 冷冻干燥后记录质量、体积、孔隙率、吸水溶胀率。
5. 推荐表征：SEM、FTIR、XRD、BET、Zeta 电位、接触角、pHpzc。

### 阶段 B：pH 梯度与藻液响应

1. 将藻液调节到 pH 4、5、6、7、8、9、10。
2. 加入等量复合材料，设置无材料空白组。
3. 在相同光照、温度、转速下振荡培养。
4. 在 0、6、12、24、48、72 h 测 OD。
5. 计算：
   - OD 增长率 = (ODt - OD0) / OD0
   - 抑制率 = 1 - 实验组增长率 / 空白组增长率
   - 若有污染物浓度，计算去除率 = (C0 - Ct) / C0 * 100%

### 阶段 C：机器学习建模

早期样本少时：

- 用响应面法或正交实验筛选主效应。
- 用线性回归/岭回归解释变量方向。

样本达到 50-100 组后：

- 用随机森林、XGBoost、SVR 或高斯过程回归预测 OD、去除率或吸附量。
- 用 SHAP 或特征重要性解释 pH、材料比例、投加量、时间等因素。

样本达到 100 组以上后：

- 建立多目标优化模型，同时优化“高去除率、低藻毒性、材料稳定性、低投加量”。

## 4. 数据表字段建议

CSV 建议至少包含：

```text
alginate_pct,chitosan_pct,acid_time_h,modifier_level,freeze_dry_h,initial_ph,initial_od,dose_g_l,rpm,time_h,temperature_c,target_od,removal_pct
```

其中 `target_od` 是模型预测目标；如果你研究污染物去除，也可以把 `removal_pct` 作为目标。

## 5. 程序说明

配套的 `experiment_predictor.html` 是一个离线浏览器工具：

- 没有数据时，使用机理启发的初始估算。
- 导入 CSV 后，会在浏览器本地训练岭回归模型。
- 可以预测终点 OD、增长率和效果等级。
- 可以自动扫描 pH、投加量和时间组合，推荐下一批实验条件。

注意：程序内置模型只适合前期实验设计和数据管理。真正论文建模应以自己的实验数据训练、交叉验证，并报告 R2、RMSE、MAE。
